# Teslo API - Nest & Postgres

1. Clone project

2. Run to rebuild node_modules

```batch
npm install
```

3. Clone file `.env.template` and rename to `.env`

4. Fill in the environment variables defined in `.env`

5. Start database

```
docker-compose up -d
```

6. Run the application in developer mode

```
npm run start:dev
```
